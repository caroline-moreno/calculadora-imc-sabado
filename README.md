# BMI Calculator

This is a simple BMI (Body Mass Index) calculator web project built using HTML, CSS, and vanilla JavaScript. The application allows users to input their weight and height, calculates their BMI, and displays the result in a user-friendly interface.

## Features

- User-friendly form for inputting weight and height.
- Calculates BMI based on user input.
- Displays the BMI result along with a simple interpretation.
- Light pink and white design for an appealing look.

## How to Use

1. Open the `index.html` file in your web browser.
2. Enter your weight in kilograms (kg) and height in meters (m).
3. Click the "Calculate BMI" button.
4. Your BMI will be displayed below the form along with a brief interpretation of the result.

## Example

- Weight: 70 kg
- Height: 1.75 m
- BMI: 22.86 (Normal weight)
